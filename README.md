# GEM HUNTER - Greenfoot Platformer 

Ένα κλασικό παιχνίδι **2D Platformer** αναπτυγμένο στο περιβάλλον **Greenfoot**. 
Ο παίκτης αναλαμβάνει τον ρόλο ενός κυνηγού θησαυρών που πρέπει να συλλέξει πολύτιμα πετράδια σε διάφορα επίπεδα δυσκολίας. 

---

## 🎮 Περιγραφή Παιχνιδιού
* **Είδος**: 2D Platform Game.
* **Σκοπός**: Συλλογή όλων των gems σε κάθε επίπεδο πριν τελειώσει ο χρόνος.
* **Περιεχόμενο**: Το παιχνίδι περιλαμβάνει 3 επίπεδα, 4 διαφορετικούς τύπους World και 5 τύπους Actor.

## 🕹️ Πώς να Παίξετε

Μόλις φορτώσει το project, πατήστε το κουμπί **Run** στο κάτω μέρος του Greenfoot και ακολουθήστε τις οδηγίες στην οθόνη:

* **Έναρξη**: Πατήστε το πλήκτρο `ENTER` στην αρχική οθόνη.
* **Κίνηση**: Χρησιμοποιήστε τα **Βέλη (Left/Right)** για να μετακινηθείτε αριστερά και δεξιά.
* **Άλμα**: Πατήστε το **Πάνω Βέλος (Up)** για να πηδήξετε πάνω σε πλατφόρμες.
* **Κίνδυνοι**: Αποφύγετε τα τέρατα (Monsters) και τα καρφιά (Spikes), καθώς θα σας κοστίσουν ζωές.
* **Νίκη**: Συλλέξτε όλα τα πετράδια (Gems) για να προχωρήσετε στο επόμενο επίπεδο.
